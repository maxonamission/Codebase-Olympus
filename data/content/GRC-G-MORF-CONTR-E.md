# ε-contracta praesens actief

## Overzicht

ε-contracta zijn werkwoorden met stam op **ε** (*ποιέω* = maken, doen). De stam-ε versmelt volgens: **ε + ε → ει**, **ε + ο → ου**, **ε + ω → ω**. Korte klinkers smelten, lange klinkers absorberen de ε.

## Paradigma

| Persoon | Ongecontract. | Gecontracteerd |
|---------|--------------|----------------|
| 1 sg. | ποιέ-ω | **ποιῶ** |
| 2 sg. | ποιέ-εις | **ποιεῖς** |
| 3 sg. | ποιέ-ει | **ποιεῖ** |
| 1 pl. | ποιέ-ομεν | **ποιοῦμεν** |
| 2 pl. | ποιέ-ετε | **ποιεῖτε** |
| 3 pl. | ποιέ-ουσι(ν) | **ποιοῦσι(ν)** |

## Contractieoverzicht voor ε

| ε + | → |
|-----|---|
| ε | ει |
| ει | ει (absorptie) |
| ο | ου |
| ω | ω (absorptie) |
| ουσι | οῦσι |

## Herkenningstips

- De 2 sg. en 3 sg. vallen op door de **-εῖ**-vormen met circumflex.
- De 1 pl. op **-οῦμεν** en de 3 pl. op **-οῦσι(ν)** verraden een ε-contractum.

## Let op

- Veel hoogfrequente werkwoorden zijn ε-contracta: *ποιέω, φιλέω, οἰκέω, καλέω, αἱρέω*.
- De infinitivus op **-εῖν** (*ποιεῖν*) komt uit ε-ειν → εῖν.
- Eén-lettergrepige stammen contracteren niet volledig (*πλέω* blijft *πλέω*, niet *πλῶ*).
