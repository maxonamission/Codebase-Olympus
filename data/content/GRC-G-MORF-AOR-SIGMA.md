# Sigmatische (zwakke) aoristus

## Overzicht

De **sigmatische aoristus** drukt een **eenmalige of afgeronde handeling in het verleden** uit. Vorming: **augment** + praesensstam + **σ** + α-uitgangen (-α, -ας, -ε(ν), -αμεν, -ατε, -αν). Modelwoord: *λύω* → *ἔλυσα*.

## Paradigma (λύω → ἔλυσα)

| Persoon | Vorm | Analyse |
|---------|------|---------|
| 1 sg. | **ἔλυσα** | ἐ- + λυ- + -σα |
| 2 sg. | **ἔλυσας** | ἐ- + λυ- + -σας |
| 3 sg. | **ἔλυσε(ν)** | ἐ- + λυ- + -σε |
| 1 pl. | **ἐλύσαμεν** | ἐ- + λυ- + -σαμεν |
| 2 pl. | **ἐλύσατε** | ἐ- + λυ- + -σατε |
| 3 pl. | **ἔλυσαν** | ἐ- + λυ- + -σαν |

## Stamtijden-rij (praesens → aor. ind. act.)

| Praesens | Betekenis | Sigmatische aoristus |
|----------|-----------|----------------------|
| λύω | losmaken | **ἔλυσα** |
| παιδεύω | opvoeden | **ἐπαίδευσα** |
| ἀκούω | horen | **ἤκουσα** |
| κελεύω | bevelen | **ἐκέλευσα** |
| θύω | offeren | **ἔθυσα** |
| γράφω | schrijven | **ἔγραψα** (φ + σ → ψ) |
| πείθω | overtuigen | **ἔπεισα** (θ + σ → σ) |

## Herkenningstips

- Zoek naar **-σα-** in de vorm: dat is vrijwel zeker sigmatische aoristus.
- 3 sg. eindigt op **-ε(ν)**, niet op -σα — niettemin blijft de σ behouden.

## Let op

- Labiale stammen (π, β, φ) + σ → **ψ**: *γράφω* → *ἔγραψα*.
- Velaire stammen (κ, γ, χ) + σ → **ξ**: *διώκω* → *ἐδίωξα*.
- Dentale stammen (τ, δ, θ) vóór σ **vallen weg**: *πείθω* → *ἔπεισα*.
