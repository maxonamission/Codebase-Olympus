# Story F2-03: Struikelpunten-overzicht per leerling

## Doel
Een mentor ziet in één oogopslag welke knopen voor een leerling
problematisch zijn: sorteerbaar op foutpercentage, op absolute
aantal-foute-pogingen en op recency.  Geen diepteklinkers meer;
per-item-details blijven in F2-02.

## Input
- `KnoopState.item_history` (F1-12)
- `KnoopState.posterior_mastery` voor kruisreferentie

## Acceptatiecriteria
- [ ] Endpoint `GET /mentor/{user_id}/struikelpunten` retourneert per
      knoop: totaal-pogingen, foute-pogingen, foutpercentage, laatste-
      datum, huidige mastery.  Default gesorteerd op recente fouten.
- [ ] Query-param `?min_attempts=3` om ruis te filteren
- [ ] Frontend-tabel met sortable columns
- [ ] Klikken op een rij linkt door naar F2-02-view

## Scope
- Alleen per-leerling (klas-aggregatie: optionele latere uitbreiding)
- Geen grafieken/heatmap — platte tabel

Medium
