# Story A5-02: Romeins dagelijks leven — familia, domus, school, eten, slavernij

## Doel
Cultuurknopen over het Romeinse dagelijks leven (SHA-C-POL-*).

## Input
Schoolmethoden (Fortuna/SPQR cultuurhoofdstukken)

## Acceptatiecriteria
- [ ] Knopen voldoen aan het ID-schema (validate_knoop_id)
- [ ] Prerequisite-edges zijn correct en vormen een DAG (geen cycli)
- [ ] JSON valideert tegen het Pydantic GraphData-model
- [ ] validate_graph.py draait zonder errors
- [ ] Beschrijving per knoop: 1-2 zinnen (kort, disambiguerend)

## Scope
Familia Romana, paterfamilias, domus/insula, ludus/school, cena, slavernij, kleding

## Geschat aantal knopen
~15
