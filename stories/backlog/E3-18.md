# Story E3-18: Content voor cultuurknopen (SHA-C)

## Doel
Didactische markdown voor de 65 gedeelde cultuurknopen in `data/graph/sha_cultuur_leerjaar1.json`. Uitleg + context + Griekse/Latijnse termen + beeldtaal (korte beschrijving voor afbeelding-placeholder).

## Verhouding tot F1-08
F1-08 levert minimale *oefenbaarheid* (items per cultuurknoop). Deze story (E3-18) levert de *didactische inhoud* die een leerling kan lezen vóór of bij het oefenen. Beide stories samen maken de cultuurknopen volwaardig — deze story kan pas draaien als F1-08 de item-structuur heeft neergezet (of minimaal gelijktijdig).

## Input
`data/graph/sha_cultuur_leerjaar1.json`, bestaande LAT-G-content als stijl, Grieks.pdf en Latijn.pdf uit `docs/` als bron.

## Acceptatiecriteria
- [ ] Geen breaking changes in bestaande models
- [ ] Alle bestaande tests blijven groen
- [ ] Minstens 60 markdowns in `data/content/` voor SHA-C-knopen
- [ ] Elk bestand: korte inleiding (1 alinea) + feitenblok + Griekse/Latijnse termen + bronvermelding waar relevant (Homerus, Livius, etc.)
- [ ] Content-ref validator (uit F1-06) groen

## Scope
~60-65 bestanden. Kan in sub-batches: E3-18a mythologie (20), E3-18b Romeins leven (15), E3-18c Grieks leven (10), E3-18d geschiedenis (15), E3-18e taal/schrift (5).
