# Story E3-18a: Content voor cultuurknopen — Mythologie

## Doel
Didactische markdowns voor de 20 mythologie-cultuurknopen (`SHA-C-MYT-*`). Elk bestand: korte inleiding, feitenblok, Griekse/Latijnse termen, bronvermelding waar relevant.

## Scope — 20 knopen

- `SHA-C-MYT-INTRO` — Klassieke mythologie (overkoepelend)
- `SHA-C-MYT-SCHEP` — Scheppingsmythen (Chaos → Titanen → Olympiërs)
- `SHA-C-MYT-OLYMP` — De twaalf Olympische goden (overzicht)
- **12 Olympiërs afzonderlijk**: ZEUS, HERA, POSEID, ATHENA, APOLLO, ARTEM, APHROD, HERMES, ARES, HEPHAI, DEMETR, HADES
- `SHA-C-MYT-HELD` — Griekse helden (overzicht)
- `SHA-C-MYT-HERAKL` — Herakles / Hercules
- `SHA-C-MYT-THESEU` — Theseus
- `SHA-C-MYT-TROJE` — De Trojaanse Oorlog
- `SHA-C-MYT-ODYSS` — Odysseus en de Odyssee

## Input
- `data/graph/sha_cultuur_leerjaar1.json` — selecteer SHA-C-MYT-* knopen
- `docs/Grieks.pdf`, `docs/Latijn.pdf` — domeinkennis
- Bestaande LAT-G-* markdowns als stijl-referentie
