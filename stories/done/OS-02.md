# Story OS-02: Type-checking toevoegen (mypy)

## Doel
Statische type-checking invoeren zodat fouten in types vroeg gevangen worden, niet pas bij runtime. De codebase gebruikt al type hints overal (CLAUDE.md vereist dat) en Pydantic; mypy maakt die hints afdwingbaar. Baseline-aanpak: nieuwe code moet schoon zijn, bestaande fouten worden geaccepteerd maar niet meer uitgebreid.

- CLAUDE.md-regel: "Gebruik type hints overal. Pydantic models voor data validation."
- Huidige `pyproject.toml` zonder mypy-config
