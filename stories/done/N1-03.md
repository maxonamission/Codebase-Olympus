# Story N1-03: Enum-klassen naar Engels

## Doel
Hernoem de resterende vernederlandste enum-*klassen* naar Engels. Alleen
de klassenamen; de enum-*waarden* (bv. `"receptief"`, `"productief"`)
blijven ongewijzigd (die zitten in data en horen bij Tier 3).

## Achtergrond
Sluit Tier 1 van epic N1 af. Niet-breaking voor opgeslagen data, want
StrEnum-waarden — niet de klassenamen — worden geserialiseerd.

## Voorgestelde renames
- `KnoopType` → `NodeType`
- `Taal` → `Language`
- `Richting` → `Direction`
- `Fase` → `Phase`
- `BloomNiveau` → `BloomLevel`
- `Bron` → `Source`
- `VerificatieMethode` → `VerificationMethod`

## Input
- `src/gymnasium_classica/models/graph.py` (enum-definities)
- alle imports/refs in `src/`, `tests/`, `scripts/`

## Acceptatiecriteria
- [x] Bovenstaande enum-klassen hernoemd op hun definitie
- [x] Alle imports en referenties bijgewerkt
- [x] Enum-*waarden* ongewijzigd (geen data-impact)
- [x] `ruff check .`, `mypy src/` en de volledige pytest-suite groen

## Scope
Alleen de enum-klassenamen. Geen waarde- of veldnaam-renames.

## Geschat
Medium — meerdere symbolen, breed verspreid.

## Resultaat
- 7 enum-klassen hernoemd in `models/graph.py` + alle referenties:
  `KnoopType`→`NodeType`, `Taal`→`Language`, `Richting`→`Direction`,
  `Fase`→`Phase`, `BloomNiveau`→`BloomLevel`, `Bron`→`Source`,
  `VerificatieMethode`→`VerificationMethod`.
- Enum-*waarden* en member-namen ongewijzigd (Tier 3). Dutch print-labels
  in scripts (bv. "Richting-verdeling:") zijn string-output, geen
  enum-referenties → buiten scope, ongemoeid gelaten.
- `ruff check .` (+ format), `mypy src/` (42), pytest (**599**) groen;
  graph-JSON parset ongewijzigd.
