# Story F1-02: Scaffolding ook in grammar-first (opt-in) bij eerste introductie

## Doel
Markdown-uitleg tonen bij de eerste keer dat een G-knoop in grammar-first-route wordt geïntroduceerd. Nu gebeurt dit alleen in context-first na een passage (`session_manager.py:476-485`), waardoor grammar-first leerlingen nooit paradigma-tabellen of herkenningstips zien.

## Open beslispunt
Defaultgedrag: altijd aan in grammar-first bij NEW_MATERIAL eerste introductie, of alleen wanneer `user.show_grammar_scaffolding` flag aan staat? Zie toelichting in epic-hoofd: aanbeveling is opt-in met default `True`, zodat E7-10 route-vergelijking (context-first vs. grammar-first) niet vertroebelt.

## Input
- `src/gymnasium_classica/api/session_manager.py:420-485`
- `src/gymnasium_classica/models/user.py` (flag toevoegen)
- F1-01 moet af zijn (rendering werkt)

## Acceptatiecriteria
- [x] Grammar-first leerling krijgt bij eerste keer dat een LAT-G-knoop met bestaande markdown wordt opgeroepen `scaffolding_content` mee
- [x] Scaffolding verschijnt niet bij herhaling van dezelfde knoop (volgende sessies / SM-2 reviews)
- [x] Context-first gedrag blijft ongewijzigd
- [x] E7-route-vergelijkingstest blijft zinvol: settings vastpinnen in testfixtures
- [x] Alle bestaande tests blijven groen; nieuwe test voor grammar-first scaffolding-trigger
