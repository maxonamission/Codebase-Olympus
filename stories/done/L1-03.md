# Story L1-03: A/B-experiment- en variant-framework

## Doel
Maak het mogelijk om leerstrategie-parameters (bijv. spacing-schema,
scaffolding-drempel, readiness-gate) **controleerbaar te variëren** per
leerling of cohort, zodat het project zijn eigen ontwerpkeuzes kan
toetsen in plaats van aannemen.

## Achtergrond
Ontwerpkeuze 12 (meetbaarheid) en de literatuur (het verschil tussen
SR-algoritmen wordt pas meetbaar bij honderden leerlingen) vragen om
een lichte experiment-infrastructuur. Geen zware platform-bouw — een
deterministische, reproduceerbare variant-toewijzing volstaat.

## Input
- `src/gymnasium_classica/models/learner.py` — `LearnerModel`
- scheduling-/SM-2-logica (`tests/test_sm2.py`, `tests/test_priority.py`,
  `tests/test_offline_scheduling.py`)
- L1-01/L1-02 (metriek + effectgrootte) voor het meten van uitkomsten

## Acceptatiecriteria
- [x] Een `ExperimentArm`/variant-concept dat een leerling deterministisch
      (op basis van user-id) aan een variant toewijst (reproduceerbaar)
- [x] Minstens één parameter is via een variant instelbaar zonder de
      kernlogica te forken (bijv. spacing-multiplier of mastery-drempel)
- [x] De variant van een leerling wordt vastgelegd zodat metriek
      (L1-01/L1-02) per variant kan worden uitgesplitst
- [x] Default-variant reproduceert exact het huidige gedrag (geen
      regressie wanneer geen experiment actief is)
- [x] Unit-tests: toewijzing is deterministisch en uitgebalanceerd;
      een niet-default variant verandert aantoonbaar het betrokken gedrag
- [x] Geen breaking changes; bestaande tests blijven groen

## Scope
Toewijzing + parametrisering + vastlegging. Geen statistische
significantie-analyse, geen frontend-toggle, geen LLM.

## Afhankelijkheden
- L1-01 en L1-02 (om uitkomsten per variant te kunnen meten)

## Geschat
Medium.

## Resultaat
- Nieuw pakket `gymnasium_classica.experiments`: `StrategyParams`
  (tunebare params, defaults == huidig gedrag), `ExperimentArm`,
  `Experiment` (deterministische, gebalanceerde SHA-256-toewijzing op
  user-id), en `strategy_params_for(learner)`.
- `ACTIVE_EXPERIMENTS` is in productie leeg → no-op (default gedrag).
- Eerste tunebare parameter: `spacing_multiplier`, doorgeprikt in
  `sm2_update` (op de vaste intervallen, geen compounding; 1.0 = identiek).
- Toewijzing vastgelegd op `LearnerModel.experiment_arms`
  (experiment-naam → arm-naam), zodat `metrics.cohort_report` per variant
  uitgesplitst kan worden (groepeer `list[LearnerModel]` op arm).
- Tests: deterministisch + gebalanceerd (400-600/1000 over 2 armen),
  niet-default variant verdubbelt aantoonbaar het SM-2-interval, default
  reproduceert baseline. **613 tests groen**, geen regressie.
