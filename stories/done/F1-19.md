# Story F1-19: Vocabulaire-items verrijken — tekstuele herkenning en productie

## Doel
Elke V-knoop in `data/graph/lat_vocabulaire_leerjaar1.json` en
`data/graph/grc_vocabulaire_leerjaar1.json` heeft vandaag exact twee items:
`luister_herkenning` en `luister_productie`.  Beide vereisen afspeelbare audio
en maken het lastig om schermloos of in een stille omgeving (bibliotheek,
klaslokaal zonder koptelefoon) te oefenen.  De scheduler kan daardoor bij
"audio uit" feitelijk niets voorleggen voor een vocabulaireknoop.

Deze story verrijkt iedere V-knoop met twee tekstuele items zodat de scheduler
altijd een receptief én productief alternatief heeft — onafhankelijk van de
audio-pijplijn — en de BKT meer signaal krijgt per knoop.

## Input
- `data/graph/lat_vocabulaire_leerjaar1.json` — 300 V-knopen, F01/F02/F03
- `data/graph/grc_vocabulaire_leerjaar1.json` — 150 V-knopen, F01/F02
- `data/vocab_sources/{taal}_{band}_words.json` — lemma/gen/mean per lemma
- `scripts/generate_luister_items.py` — referentie-patroon (dict-stimulus,
  item-ID-nummering, distractor-pool)
- `scripts/content_coverage.py` — dekkingsrapport (F1-11)

## Acceptatiecriteria
- [x] Nieuw script `scripts/generate_vocab_enrichment.py` dat per bestaande
      V-knoop in de twee vocabulaire-JSON's tot twee nieuwe items toevoegt:
  - `herkenning` (richting `receptief`): lees het lemma in de bron-taal, kies
    de juiste Nederlandse vertaling uit vier opties.  Stimulus is een dict
    met `instruction`, `lemma`, `options`.
  - `productie` (richting `productief`): gegeven de Nederlandse vertaling,
    typ het lemma in Latijn of Grieks.  Stimulus is een dict met
    `instruction`, `translation`.
- [x] Idempotent: een tweede run voegt geen duplicaten toe (het script
      controleert op bestaand `type` per knoop, net als
      `generate_luister_items.py`).
- [x] Item-ID's volgen `ITEM-{KNOOP_ID}-{NR:03d}` en sluiten aan op de
      bestaande nummering (`next_item_nr`-patroon).
- [x] Distractor-pool wordt per bestand opgebouwd uit de andere V-knopen in
      datzelfde bestand; correct antwoord uitgezonderd; deterministisch via
      `random.seed(42)`.
- [x] Lemma-extractie gebruikt dezelfde helpers als
      `generate_luister_items.py` (`parse_titel`, `extract_first_word`,
      `extract_short_translation`) — liefst herbruikt of kopie met identieke
      gedragsregel voor Latijn/Grieks.
- [x] IRT-parameters: `moeilijkheid_initieel` uit `U(-0.8, 0.2)` voor
      herkenning en `U(0.3, 1.8)` voor productie; `discriminatie_initieel =
      1.0`; `verwachte_tijd_sec = 10` (herkenning) / 20 (productie).
- [x] `bron = "llm_gegenereerd"`, `audio_ref` **niet** gezet (tekstueel item).
- [x] Script valideert achteraf via `GraphData(**data)` en schrijft het
      bestand terug met UTF-8 zonder BOM en trailing newline.
- [x] CLI-samenvatting toont per bestand: knopen bijgewerkt, items toegevoegd,
      totaal items per type; geen volledige JSON op stdout (zie CLAUDE.md).
- [x] `python -m pytest tests/ -q` blijft groen (in bijzonder
      `test_content_coverage.py`, `test_graph_loader.py`, `test_vocab_loader.py`).
- [x] `python scripts/content_coverage.py` draait na afloop en toont dat de
      `items`-dekking voor V-knopen op 100 % blijft (geen regressie).

## Scope
- Alleen de twee vocabulaire-JSON-bestanden; **geen** wijzigingen in
  grammatica-, alfabet-, cultuur- of transfer-bestanden.
- **Geen** nieuwe content-markdown, **geen** audio-generatie (de nieuwe items
  zijn tekstueel).
- **Geen** wijzigingen aan Pydantic-modellen, loader of scheduler — de
  bestaande `ItemType.HERKENNING`/`PRODUCTIE` enums worden hergebruikt.
- Geen conflict met Golf 1A (`grc_alfabet.json`) of Golf 1B (`data/content/`).

## Niet-doel
- Geen `contextueel`-items met voorbeeldzinnen (aparte vervolgstory — vraagt
  handmatige zinsvoorbeelden per lemma).
- Geen `analyse`-items (voor vocabulaire weinig informatief).
- Geen frontend-wijzigingen; de UI rendert `herkenning`/`productie` al.

## Afhankelijkheden
- F1-11 (dekkingsrapport) — voor verificatie na afloop.
- A4-* (vocabulaire-epic) — levert de bestaande knopen + `luister_*`-items.

## Geschat
Klein (één script + ~900 nieuwe items, volledig gegenereerd).
