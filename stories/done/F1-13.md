# Story F1-13: Mentor-view pagina (capstone Sprint 5)

> Hernummerd binnen Sprint 5: dit nummer stond in EPICS al gereserveerd
> als "Frontend mentor-view (placeholder)". De oorspronkelijke tooltip-
> scope is verhuisd naar F1-20.

## Doel
Een ingelogde mentor krijgt één pagina (`/mentor`) die de in deze sprint
gebouwde diagnostiek samenbrengt: kies een gekoppelde leerling → zie hun
struikelpunten → klik door naar de letterlijke pogingen per knoop.  Dit
maakt F2-02 en F2-03 voor het eerst echt bruikbaar en realiseert de
sprint-DoD: *"een mentor kan inloggen, leerlingen koppelen en per
leerling per knoop zien wáár en hóe het misgaat."*

## Input
- `components/MentorStruikelpunten.jsx` (F2-03) en `MentorAttemptList.jsx`
  (F2-02) — al gebouwd en getest
- `api.getMentees / getStruikelpunten / getMentorAttempts`
- `App.jsx` (react-router, ProtectedRoute) en bestaande pagina-stijl
  (`pages/Dashboard.jsx`)

## Acceptatiecriteria
- [x] Nieuwe beschermde route `/mentor` met een `MentorView`-pagina
- [x] Pagina haalt de mentees op en toont ze als selecteerbare lijst
- [x] Een mentee kiezen laadt diens struikelpunten in de
      `MentorStruikelpunten`-tabel
- [x] Een struikelpunt-rij aanklikken laadt de laatste pogingen op die
      knoop in de `MentorAttemptList` (doorklik F2-03 → F2-02)
- [x] Laad- en foutstatus afgehandeld; lege staten netjes getoond
- [x] Vitest-test die de flow met gemockte api dekt (mentees →
      struikelpunten → pogingen)

## Scope
- Frontend-compositie van bestaande componenten; geen nieuwe endpoints
- Geen mentor-login/registratie-UI (rol wordt via CLI gezet, F2-01); de
  pagina gaat uit van een al-ingelogde mentor
- Geen klas-aggregatie of grafieken

## Resultaat
- `pages/MentorView.jsx` — drie-staps-compositie: mentee-lijst →
  `MentorStruikelpunten` (F2-03) → `MentorAttemptList` (F2-02), met
  `loading`/`error`/lege-staat-afhandeling.
- Beschermde route `/mentor` toegevoegd in `App.jsx`.
- `pages/MentorView.test.jsx` — 5 vitest-tests met gemockte `../api`
  (mentee-lijst, selectie laadt struikelpunten, rij-klik laadt pogingen,
  lege staat, foutafhandeling).
- Mentor-pagina-stijlen in `styles/main.css`.

**Status:** frontend 47 vitest-tests groen, `vite build` slaagt, eslint
