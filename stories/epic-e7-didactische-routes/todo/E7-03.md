# Story E7-03: Eerste set leespassages — Latijn onderbouw

## Doel
Schrijf 20 korte leespassages voor Latijn leerjaar 1: aangepaste teksten die geleidelijk complexer worden. Elke passage oefent 2-4 grammaticaknopen en 5-10 vocabulairewoorden.

## Input
Fortuna/SPQR tekstfragmenten als inspiratie (niet kopiëren), A1 grammaticaknopen, A4 vocabulaire

## Acceptatiecriteria
- [ ] Geen breaking changes — bestaande grammar-first route blijft default
- [ ] Alle bestaande tests blijven groen
- [ ] Nieuwe functionaliteit heeft eigen tests

## Scope
20 passages oplopend in complexiteit: 1-2 naamvallen → alle naamvallen + presens → imperfectum/perfectum + bijzinnen. Per passage: brontekst, woordannotaties, gekoppelde knoop_ids.

## Geschat
~20 passages
