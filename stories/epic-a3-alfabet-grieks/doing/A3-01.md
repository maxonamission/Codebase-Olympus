# Story A3-01: Letters herkenning — 24 letters, majuskel + minuskel

## Doel
De 24 letters van het Griekse alfabet als individuele of gegroepeerde knopen.

## Input
Standaard Grieks alfabet, Pallas les 1

## Acceptatiecriteria
- [ ] Knopen voldoen aan het ID-schema (validate_knoop_id)
- [ ] Prerequisite-edges zijn correct en vormen een DAG (geen cycli)
- [ ] JSON valideert tegen het Pydantic GraphData-model
- [ ] validate_graph.py draait zonder errors
- [ ] Beschrijving per knoop: 1-2 zinnen (kort, disambiguerend)

## Scope
α-ω, majuskel+minuskel, naam+klank, gegroepeerd naar visuele gelijkenis met Latijn

## Geschat aantal knopen
~24
